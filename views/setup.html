  5. Find the row with name token. Click on it and copy the entire Value field (long string, usually starts with eyJ…).

Lets BambuBoard set the broadcast title, description, privacy and the “made for kids” flag (the “Connect account” path, like OBS). Skip this to just paste a stream key on the Live page instead. How to create a Google OAuth client ↗

  1. Open the Google Cloud Console and create (or pick) a project.
  2. APIs & Services → Library → enable the YouTube Data API v3.
  3. OAuth consent screen: External, add yourself as a Test user (no Google review needed while in “Testing”).
  4. Credentials → Create credentials → OAuth client ID → Web application.
  5. Under Authorized redirect URIs, add the exact URI shown below.
  6. Copy the Client ID and Client secret into the fields below and Save.
