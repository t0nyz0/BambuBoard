BambuBoard composites your layout — camera and every widget — into one page. Add it to OBS as a single Browser Source, or open it anywhere. Edit the layout and hit 🔴 Go Live; OBS updates on its own.

Add to OBS — one Browser Source

 

Download OBS scene
  1. In OBS: + → Browser, paste the URL above, set the size to your canvas (e.g. 1920×1080). That's it — no camera source, no SDP.
  2. Prefer importing? Use the Download OBS scene button, then Scene Collection → Import — it's just that one Browser Source.
  3. Design your overlay in the Layout editor and hit 🔴 Go Live — this page and OBS update automatically.

Stream to YouTube (beta — no OBS)

Connect your YouTube account to set the broadcast title, description, privacy and the “made for kids” flag — just like OBS. First add a Google OAuth client in Setup → YouTube streaming.

Connect your YouTube account, then set the title/description/privacy and go live straight from your browser — no OBS, no manual stream key.

Connect YouTube account

One-time Google sign-in. Do this from the machine running BambuBoard (open it at http://localhost:8080) — Google only allows the sign-in redirect on localhost.
